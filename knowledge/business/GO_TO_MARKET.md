# GO_TO_MARKET — 初期獲得戦略

## 役割
最初の10院を獲得し、製品の価値仮説を検証するための戦略を定義する。
「誰に・何を・どう伝えるか」の行動指針とする。

## 管理対象
- ターゲット院の優先順位基準
- 営業アプローチと訴求メッセージ
- 初期10院の検証指標

## 書いてよい内容
- 獲得チャネルと優先順位
- 初回接触〜契約までのフロー
- 検証指標の定義と合格水準

## 更新ルール

- 訴求メッセージの反応が悪い場合は理由を追記して差し替える

## ターゲット院の優先順位基準

以下の条件を多く満たすクリニックを優先する。

| 条件 | 理由 |
|---|---|
| 美容皮膚科・歯科・婦人科 | 単価高め・キャンセル損失が数値化されやすい |
| 月間無断キャンセル 5件以上 | 課題が顕在化しており動機が強い |
| 院長がSNS発信 / Web集客に積極的 | ITリテラシー中以上。意思決定が早い |
| 既存予約システムがSTORES予約 / Google カレンダー | 連携導入を検討しやすい |
| 受付スタッフ 1〜3名 | オンボーディングの工数が小さい |
| 自費診療の比率が高い | 保険点数に縛られずキャンセル料設定が自由 |

## 獲得チャネルと優先順位

### チャネル 1: 知人・紹介（最優先）
- 院長・クリニック経営者の知人経由
- 医療系SNS（X・Facebookグループ）でのクチコミ
- **目標**: 最初の 3院 は紹介経由で獲得する

### チャネル 2: 院長向けコミュニティ
- 医師向け勉強会・経営セミナー（自費診療 / 美容 / 歯科系）
- 参加・登壇してデモを見せる
- 「無断キャンセル対策」をテーマにした事例発表

### チャネル 3: ダイレクト DM / コールド
- 美容皮膚科・歯科のクリニックに対してInstagram・Webサイトのフォームから接触
- 訴求メッセージを絞って反応率を計測する
- **注意**: 10院中 2〜3院程度の補完チャネルとして使う

---

## 初回接触〜契約フロー

```
1. 訴求メッセージで接触
         ↓
2. 30分オンラインヒアリング（月間損失額を一緒に計算する）
         ↓
3. デモ環境で実際のフロー（SMS送信〜同意〜カード登録）を体験
         ↓
4. 成功報酬型（案1）で 1ヶ月無料トライアル提案
         ↓
5. オンボーディング（予約情報 CSV 受取 → 初回 SMS 送信設定）
         ↓
6. 最初の実績（1件のキャンセル料回収）を確認してから本契約へ
```

## 訴求メッセージ

### ファースト訴求（接触時）
> 「無断キャンセルされたとき、キャンセル料を請求できていますか？
> 言いにくいから諦めている、というクリニックのために作りました。」

### 数値訴求（ヒアリング後）
> 「月10件のキャンセル × 単価15,000円 = 月150,000円の損失。
> 回収できた分の15%だけいただきます。初期費用ゼロです。」

### 安心訴求（懸念への対応）
> 「患者さんが事前に同意しているので、受付が請求する必要はありません。
> システムが自動で請求します。」

## 検証指標（初期10院）

### 導入前の検証
| 指標 | 目標水準 | 計測方法 |
|---|---|---|
| ヒアリング→デモ同意率 | 50%以上 | 接触件数 / デモ依頼件数 |
| デモ→トライアル同意率 | 60%以上 | デモ件数 / トライアル開始件数 |
| 平均ヒアリング所要時間 | 30分以内 | 実測 |

### 導入後の検証
| 指標 | 目標水準 | 計測方法 |
|---|---|---|
| 患者カード登録率 | 70%以上 | カード登録数 / SMS送信数 |
| キャンセル料回収率 | 80%以上（回収試行のうち） | captured数 / 請求試行数 |
| 月間回収額（クリニック平均） | 30,000円以上 | DBから集計 |
| 3ヶ月継続率 | 80%以上 | 3ヶ月後の稼働院数 / 開始院数 |

### 解約アラート指標
| 指標 | アラート水準 |
|---|---|
| 患者カード登録率 | 50%を下回ったら即ヒアリング |
| 月間キャンセル件数 | 0件が2ヶ月続いたら「低単価リスク」判定 |
| クリニックからの問い合わせ頻度 | 週2件超えは定着リスク |

## 失敗パターンと対策

| 失敗パターン | 対策 |
|---|---|
| 「患者がカード登録を嫌がる」と言われる | 登録率データを見せ、UX改善の優先度を上げる |
| 「効果が出る前に解約される」 | トライアル期間中に1件でも回収実績を作る |
| 「受付が使いこなせない」 | オンボーディングを1対1・30分で完結させる |
| 「Stripeの審査が通らない」 | 事前に医療系事業者の審査ポイントを把握して案内する |
